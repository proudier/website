Did you know it is possible to use a standard pager, such as less, within the MySQL command line tool?
Using the pager, you can scroll throught the result of your queries more easilly.

Setting the pager
[sql toolbar=false]
mysql> pager less
PAGER set to 'less'
[/sql]

Testing
[sql toolbar=false]
mysql> use mysql
mysql> show tables
+---------------------------+
| Tables_in_mysql |
+---------------------------+
| columns_priv |
| db |
| event |
| func |
:
[/sql]

Reminder: press the 'q' key to escape from the pager.

Disabling the pager
[sql toolbar=false]
mysql> nopager
PAGER set to stdout
[/sql]

Afterword
These features are only available on GNU/Linux.
