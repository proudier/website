nSquare

nSquare was a monolitic kernel, running in 32bits ("protected mode") on the x86 platform. It embedded a keyboard and a VGA console driver.

It was able to load executables from memory and start them as processes. The CPU time was then shared in a round-robin fashion. The bootstrap process was based on Grub.

The debugging was done by running the kernel inside Bochs, an IA-32 emulator.

Sadly, I lost the source sometime ago because of a system crash (yeah, yeah, I do backups since then).

A must read on the topic: "Operating Systems: Design And Implementation" by Andrew S. TANENBAUM.

Technology: x86 assembler, C.

Pierre's Lib

The development of nSquare required a lot of documentation and looking for it took some time (search enginesGoogle were not as good as today). So I decided to share what I found through a website: Pierre's Library.

I had totally forgotten about this site for a while. And I was very surprised when I found it was still online!
